This is a personal learning project built to practice and understand the fundamentals of Laravel development while integrating it with React using Inertia.js.

The project implements a simple Asset Management System where users can create and manage assets with details such as name, model, quantity, price, description, and images.

The goal of this project is to explore and practice Laravel concepts such as:

Form Requests for validation

DTO (Data Transfer Objects)

Service Layer

Repository Pattern

File uploads

Laravel + React integration using Inertia.js

🛠 Tech Stack

Backend: Laravel

Frontend: React + Inertia.js

Database: MySQL
